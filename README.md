
# Hangul Practice

[https://hangul-practice-production.up.railway.app/](https://hangul-practice-production.up.railway.app/)

This project was created as a tool to practice Hangul (Korean writing).

Technology used:
 - Web Speech API to access text to speech
 - HTML Canvas to capture user drawn image
 - Google Cloud Vision API as an OCR to compare the user drawn image to the correct Korean text
